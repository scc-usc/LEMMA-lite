💡 LEMMA Forecast — Help

A browser-based Hubverse forecast generator. Everything runs entirely in your browser — no server and no internet connection required. Generate multiple versions of simple prediction models and combine them using simple ensemble or a Random Forest

Features

  • Upload a Hubverse target-data CSV (plus an optional location/population CSV; recommended for better accuracy).
  • Approaches: Flatline and ARIMA. Ensembles: Basic (quantile) and Quantile Random Forest.
  • Multi-target selection when the data has a target column.
  • Training window (for Random Forest) & forecast origins set with the dual sliders under the plot.
  • Interactive chart — zoom (wheel / drag), pan (Ctrl + drag), and click any point to set the forecast origin.
  • Download the result as a Hubverse quantile CSV.

Quick start

  1. Upload your Hubverse target data CSV.
  2. (Optional) upload a location / population CSV to convert counts to rates.
  3. Choose the target(s), approach, ensemble, weeks ahead, and quantiles.
  4. Adjust the training window (for Random Forest) and forecast origin sliders.
  5. Click Generate Forecast, review the chart, then Download CSV.

Input format

Hubverse target data CSV (one observation per location per week):

ColumnRequiredNotes
locationYesLocation code/name.
target_end_date or dateYesWeekly date of the observation.
observation / weekly_rate / valueYesThe observed value.
targetOptionalIf present, choose target(s) in the UI; if absent, all data is treated as a single implicit target.

Population CSV (optional): columns location_name and population.

Output

A Hubverse quantile forecast with columns: origin_date, horizon, location, target (multi-target runs only), output_type (quantile), output_type_id (the quantile level), and value.
